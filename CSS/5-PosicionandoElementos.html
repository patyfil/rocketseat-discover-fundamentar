3 - Position (Posicionamentos)

Controlar onde, na página, o elemento irá ficar, alterando o fluxo normal dos elementos.

- Name: Position
- Value: static | relative | absolute | fixed

Lembrando que o fluxo normal dos elementos é ficar um abaixo do outro, a não ser no caso de elementos inline, que ficam um ao lado do outro.

<div class="box1 box"></div> <div class="box2 box"></div> <div class="box3 box"></div> .box { width: 50px; height: 50px; margin-bottom: 8px; }

4 - Static

Os elementos posicionados estáticos não são afetados pelas propriedades superior, inferior, esquerda e direita.
Um elemento with position: static; não está posicionado de nenhuma maneira especial; está sempre posicionado de acordo com o fluxo normal da página

.box1 { background: red; } .box2 { background: green; } .box3 { background: blue; }

5 - Relative

position: relative;

5 propriedades: top, right, bottom, left, z-index

Um elemento com position: relative; é posicionado em relação à sua posição normal.
Definir as propriedades superior, direita, inferior e esquerda de um elemento relativamente posicionado fará com que ele seja ajustado fora de sua posição normal.
Outros conteúdos não serão ajustados para caber em qualquer lacuna deixada pelo elemento.

.box4 { background: red; position: relative; left: 40px; top: 80px; } .box5 { background: green; } .box6 { background: blue; }

6 - Absolute

position: absolute;

5 propriedades: top, right, bottom, left, z-index

É como se criasse outra camada por cima da página, e o 'box7 vermelho' estivesse sozinho na página

Um elemento com position: absolute; é posicionado em relação ao ancestral posicionado mais próximo (em vez de posicionado em relação à janela de visualização, como fixo).
No entanto; se um elemento posicionado de forma absoluta não tiver ancestrais posicionados, ele usará o corpo do documento e se moverá junto com a rolagem da página.
Nota: Elementos posicionados de forma absoluta são removidos do fluxo normal e podem se sobrepor a elementos.

div { position: relative; } .box7 { background: red; position: absolute; // em relação a div que é relative right: 0px; top: 0px; z-index: 1; } .box8 { background: green; } .box9 { background: blue; }

7 - Fixed

Um elemento com position: fixed; é posicionado em relação à janela de visualização, o que significa que ele permanece sempre no mesmo lugar mesmo se a página for rolada.
As propriedades superior, direita, inferior e esquerda são usadas para posicionar o elemento.
Um elemento fixo não deixa espaço na página onde normalmente estaria localizado.

8 - Sticky

Um elemento com position: sticky; é posicionado com base na posição de rolagem do usuário.
Um elemento fixo alterna entre relative e fixed, dependendo da posição de rolagem.
Ele é posicionado em relação até que uma determinada posição de deslocamento seja atingida na janela de visualização - então ele "gruda" no lugar (como position:fixed).

Nota: O Internet Explorer não suporta posicionamento fixo. O Safari requer um prefixo -webkit-.
Você também deve especificar pelo menos um valor para top, right, left ou bottom para que o posicionamento fixo funcione.

       

9 - Element Stacking

.div { height: 100px; } .box10 { background: red; position: absolute; top: 5px; left: 5px; } .box11 { background: green; position: absolute; top: 10px; left: 10px; } .box12 { background: blue; position: absolute; top: 15px; left: 15px; }

10 - Flexbox

* Nos permite posicionar os elementos dentro da caixa. * Controle em uma dimensão (horizontal ou vertical) * Alinhamento, direcionamento, ordenar e tamanhos

div.parent { display: flex; }

Flex-direction

* Qual a direção do flex: horizontal ou vertical

row | column

Alinhamento

justify-content
align-items

HTML

<div class="container"> <div class="box blue"></div> <div class="box red"></div> <div class="box green"></div> </div>

CSS

.container { display: flex; justify-content: space-between; align-items: center; width: 30%; } .box { width: 50px; height: 50px; margin-bottom: 0px; } .blue { background-color: blue; } .red { background-color: red; } .green { background-color: green; }

    <header>
      <div>logo</div>

body { margin: 0; height: 100vh; display: grid; grid-template-areas: "header header" "main aside" "footer footer"; grid-template-rows: 50px 1fr 50px; grid-template-columns: 1fr 250px; font-size: 24px; } header { grid-area: header; background-color: green; display: flex; justify-content: space-between; align-items: center; padding: 0 20px; } main { grid-area: main; background-color: red; } aside { grid-area: aside; background-color: blue; } footer { grid-area: footer; background-color: gray; }
