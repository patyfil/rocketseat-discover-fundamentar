    9 - Font Stretch

    Alargamento ou encolhimento da fonte
    Aceita palavras-chaves como: expanded, condensed, normal
    Aceita porcentagens de 50% a 200%
    Essa propriedade não vai funcionar em todas as fontes 

        p {
      font-stretch: expanded;
    }
    Referência

    12 - Line Height

    Espaços entre linhas

    Pode ser com unidades ou sem unidades de medida

    Comuns: 1.5 ou 2

        p {
      line-height: 1.5;
    }
    Referência

    13 - Text Transform

    Transformação do texto
    Valores podem ser: none | capitalize | uppercase | lowercase | full-width | full-size-kana

        p {
      text-transform: uppercase;
    }
    Referência

    14 - Text Decoration

    Aparência decorativa de um texto

    Valores line: none | underline | overline | line-through | blink

    Valore Style: wavy | dotted | double | dashed | solid

    Valores Color: <color> values

        p {
      text-decoration: underline dashed red; /* shorthand */
    }
    Referência

    15 - Text Align

    Alinhamento de um texto

        p {
      text-align: center; /* left | right | center | justify */
    }
    Referência

    16 - Text Shadow

    Sombra aplicada a um texto

    Permite múltiplos valores

        p {
      text-shadow: 1px 1px 1px red,
                   2px 2px 1px blue;  /* offset-x | offset-y | blur-radius | color */
    }
    Referência

    17 - Shorthand

    font-style, font-variant, font-weight, font-stretch, font-size, line-height, e font-family.

      font: italic small-caps bold 12px/1.5 arial, sans-serif;
